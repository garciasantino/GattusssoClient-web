FAQ

1.¿Porque la V6 funciona unicamente en TLauncher?

Esto sucede porque el launcher original de minecraft no puede descargar el file de la version usada de Forge (lo que lo compone) ya que no lo puede leer localmente, mientras que TLauncher si. Ya que el file al estar alojado en una pagina para el launcher premium y no de forma local, contiene varios errores que hace que no pueda ejecutar el juego. Hasta ahora no se encontro una solucion.

2.No me abre ninguna version en PojavLauncher, ¿que hago?

Intenta descargando la JDK 17 para Android y ponerlo en un archivo zip, para luego cargarlo en la configuracion del launcher, reemplazando asi la JDK 8 por la 17 en el modo de seleccion.

3.¿Porque la V6 & V5 no funcionan para Pojav?

Esto es porque requieren de arguments de Java para que funcionen, en el cual no se puede compartir el codigo por un problema de inactividad con el propietario de aquello. Asi que habra que esperar a su respuesta para que el codigo sea compartido como su Solucion. Mientras que esta version ya viene de por si con el codigo integrado sin la necesidad de configurar nada del launcher para su funcionamiento.
